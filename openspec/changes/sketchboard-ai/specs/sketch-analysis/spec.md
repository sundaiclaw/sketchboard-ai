# Spec: Sketch analysis and AI UI planning

## Requirements

### Requirement: upload and summarize sketch inputs
The system shall allow the user to upload a sketch artifact and produce a human-readable structural summary.

### Requirement: AI concept generation
The system shall use an OpenRouter free model to generate a product concept, UI copy, and structured layout plan from the sketch summary.

### Requirement: live mock preview
The system shall render a preview screen from the AI-generated structured layout.

### Requirement: demoability
The system shall include a built-in sample so a reviewer can test the full flow in under a minute.
